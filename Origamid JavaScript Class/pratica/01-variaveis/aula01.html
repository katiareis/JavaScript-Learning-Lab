Variáveis em JavaScript

Em programação, variáveis são espaços que são utilizados para guardar dados(valores) na memória.

Variáveis ajudam a evitar a repetição desnecessária de código.

Podem ser utilizadas para fazer valor o Conceito DRY (Don't repeat yourself)

Declarando uma variável

Há três formas de declarar uma variável em JavaScript

  • Usando a palavra reservada var
  • Usando a palavra reservada let
  • Usando a palavra reservada const

Sintaxe

PalavraReservada + nomeDaVariável = Valor

          
          var x = 5;
          var y = 6;
          var z = x + y;

Sobre o exemplo acima:

  • A variável x armazenou o valor 5 na memória
  • A variável y armazenou o valor 6 na memória
  • A variável z armazenou o valor 11 na memória

Temos algumas regras para criar nome de variáveis:

  • Não pode começar com número
  • Não pode ter caracteres especiais
  • Não pode utilizar acentos
  • Não pode utilizar espaços
  • Deve começar com letra minúscula e seguir o padrão camelCase, onde cada palavra é iniciada com maiúsculas e unidas sem espaços

É muito importante utilizar nomes significativos nas variáveis, para facilitar a identificação
